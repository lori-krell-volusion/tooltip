Tooltip Help and Info

Add information as you need for working on Tooltips for the Volusion Admin.

File Naming

File naming structure should be:

module_pagename_fieldname

Example: Customer_Accounts_Name

Some tooltips are located deeper in the pages via tabs, etc. These names should follow a similar path:

Example: Inventory_ImportExport_VolusionAPI_GoogleShoppingFeed_FTPUsername

Note: Anything that redirects you to the Config Variables would be Settings_ConfigVariables_fieldname

Folder Locations

All tooltips will be created within Module folder / Tooltips

Styling for Content

For DATATYPE, have a space before and after the colon. For example: STRING : 35
