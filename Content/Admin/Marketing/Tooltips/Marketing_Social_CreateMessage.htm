Create Message

  

In this field, type the message you want to post to Facebook and / or Twitter.

If you accessed this field using the Share button on a product or category edit page, a shortened version of its URL is automatically inserted. Be careful not to alter it. If you accessed this field from Marketing > Social, you can search for products and / or categories to include using the functions below.

Note that if you create a Facebook or Twitter post that contains a link to a temporary product and that product expires, the link will produce a 404 error.
