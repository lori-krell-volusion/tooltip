Create a Banner

To create a text message for your "Like" promotion, select Text and type the text you want to display.

To upload a banner image for your "Like" promotion, select Image and click the Upload button to locate the file on your computer. It can be any size, but if it is wider than 755 pixels, it will be automatically re-sized. You can use.jpg, .gif, or .png format.

If you have already upload an image it,a Delete button displays that allows you to remove the file.
