Payment Info Delete Event

In this field, you can type the following values:

AUTHORIZE - This value deletes credit / debit card data immediately after the payment has been successfully authorized. For fully-supported gateways, merchants can still capture, credit, debit, and void transactions once an authorization has been obtained. This is the default and most secure setting, and the one that Volusion recommends.

CAPTURE - This value deletes credit / debit card data after authorized funds have been captured. Some gateways require this value in order to function with Volusion.

SHIP - This value deletes credit / debit card data after the order status changes to Shipped. A small number of integrated gateways may require this value. Volusion only recommends only using it if absolutely necessary.

NEVER - This value deletes credit / debit card data until the number of days specified in the Payment Info Max Days To Keep field have passed. This value is the least secure option and we do not recommend using it.
