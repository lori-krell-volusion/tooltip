Payment Info Max Days To Keep

  

If you have typed "NEVER" in the Payment Info Delete Event field, the number you type in this field determines the number of days that pass before credit / debit card data is deleted.

If you set this value to a number higher than 14, customers will be given a Keep my payment method on file for easy checkout on future orders option during checkout.

If you set this value to zero, credit / debit card data is still collected during the checkout process and stored temporarily in the database. If the shopper abandons the purchase before completion, the card data is deleted within 48 hours.
