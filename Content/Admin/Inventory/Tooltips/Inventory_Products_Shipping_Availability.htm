Select an option from the drop-down menu to inform shoppers of an estimated delivery time. Note that you must go to Settings > Config Variables > Search, type "availability" in the Name field, click the orange Search button at the bottom, and select the Display Shipping Availability check box.

Select the Add Custom Text check box below to create type your own custom availability statement.
