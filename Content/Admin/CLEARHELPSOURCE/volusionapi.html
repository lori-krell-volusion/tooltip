The Volusion API

Documentation

The Volusion API allows you to share your Volusion data with third-party services to extend the capabilities of your store. Using the Volusion API, you can integrate with industry-standard financial software, award-winning order management services, and even create your own custom integrations using XML.

Note that to obtain access to the Volusion API, you must have a Volusion Gold store account or above. If you have a Steel, Bronze, or Silver store account, you can upgrade by logging in to your myVolusion Area.

There are several features of the Volusion API that you can access on this page. Below, you'll find a description of each section. Click on any available link to view that API section settings and Instant Help article for more information.

Enable public XML for Featured Products

This is a special XML export that can be generated using the Volusion API. This will output information for all Featured Products into an XML file once a day.

Select Enable public XML for Featured Products and click Save. This export is performed every 24 hours. You can reset the daily timeout by clicking the Reset button.

To access the featured product export's XML file directly, you can open your web browser to the following URL:

http://www.yourdomain.com/net/webservice.aspx?api_name=generic\featured_products

Be sure to substitute ""www.yourdomain.com"" in the above URL with your store's domain name.

Enable public XML for All Products

Similar to the Featured Products export, this can be used to generate an XML file that contains information for every product within your Volusion store's database. This export can be used in a variety of ways to share your store's product information with third-party software and services utilizing XML.

Select Enable public XML for All Products and click Save. As with the Featured Products Export, this export is generated once every 24 hour period. You can override this by clicking the Reset button.

To view your store's All Products export, visit the following URL:

http://www.yourdomain.com/net/webservice.aspx?api_name=generic\all_products

Be sure to substitute ""www.yourdomain.com"" in the above URL with your store's domain name.

Note that as this export will contain information for all products within your store, generating it will be resource-intensive if your store has a large number of products.

Quickbooks Web Connector

The Volusion API enables you to integrate your Volusion store with Intuit's QuickBooks software using the QuickBooks Web Connector software. This provides a more robust integration with Volusion than the standard IIF file export. Click on the Manage Quickbooks Web Connector HERE link for more information and begin configuring your store to integrate with Intuit's QuickBooks Web Connector (QBWC).

Generic

This section contains three options that allow you to export a list of your Volusion store's orders, products, or customers from your store's database to an XML file. This can be used to integrate your store's data with various third-party services and applications.

Additionally, you can click on the Google Merchant Center Integration link to configure your store to integrate with Google's Google Merchant Center ecommerce marketing service.

Order Management

You can use the exports provided within this section to integrate your online store with Stone Edge Order Manager (SEOM). Stone Edge is a leader in ecommerce and business logistic software, specializing in order fulfillment and management processes. Click any of the three options to create an XML document specially formatted for use with SEOM.

Here you are given the option to export customer, order, and product data from your Volusion store to an XML file for use with Stone Edge's software.

Shipping

This section of the Volusion API allows you to create exports which can be used to integrate your online business with shipping services and software from the US Postal Service, Pitney Bowes, and Stamps.com.

The Endicia DAZzle export can be used to generate an XML file containing order information from your online store that can be imported into Endicia's DAZzle shipping label and postage management software. Endicia's software and service is for use with the United States Postal Service. Click the Run button to enter the Endicia page and find out more on how to use this portion of the Volusion API.

The Pitney Bowes export can be used to generate an XML file containing order data which can then be used to integrate your Volusion store with products and services from Pitney Bowes.

The Stamps.com export can be used to generate an XML file containing order data which can then be used to integrate your Volusion store with products and services from Stamps.com.
