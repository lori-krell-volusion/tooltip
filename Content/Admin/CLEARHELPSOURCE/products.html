Common Questions

How do I create a product?

  1. Click the Add button.
  2. Fill in the required product details - Name, Code, Price and Weight. You can also enter description fields, upload images, and add further information as desired.
  3. Click Save, and you've created a product!

See Learn More for information on all aspects of setting up products.

How do I edit a product?

  1. In the Product Code column, click the required product code.
  2. You can now edit the Product page as required.
  3. Click Save

See Learn More for information on all aspects of editing products.

How do I add a product to a category?

  1. Select the required categories from the Categories list.

If you need to create a new category:

  1. Click Create New Category and enter the new Category Name.
  2. If the new category is to be a subcategory, select the parent category from the Subcategory of list.
    If not, select the navigation menu in which to display your category from the Show in Menu dropdown.
  3. Click Save.

How do I add options to a product?

  1. Click Add Options.
  2. Click on the Option Category containing the required options and select the Options you required.
  3. You can create a new option in an Option Category by clicking Add New Option.
  4. If you want to create a new option category, scroll to the bottom of the list of Option Categories and click Add New Option Category.
    Enter the new Option Category name and click Save.
  5. Click Save on the popup.

How do I edit multiple products at once?

When you have many products to update, you can edit information directly on the All Products page, instead of having to open each product's page. Editable columns show an Edit option when you hover over their headers. Be sure to click Save before moving to the next page.

You can customize which columns are displayed by clicking Settings > Customize Columns and selecting the desired columns. You can customize columns and edit column contents this way in most Volusion tables.

How do I manage products with many options?

Suppose you sell a dress that comes in 6 different sizes and 6 different colors. By using options, you can create a product with 36 different option combinations. For inventory tracking purposes, however, you might prefer to treat these as 36 separate child products, each with its own product code, stock status, and other details.

To create child products from a product with options, select Enable Options Inventory Control and click Save.

These child products will all have unique product codes, but you can manage them from the parent product's Options Inventory Control. See Using the Inventory Control Grid for more information.

Using the Products Pages

The All Products Page

The All Products page gives you access to all the products in your store, and lets you add new products.

You can search for a product by clicking Search and entering any information you know, such as Code, Name or Price. Click the orange Search button at the bottom of the page to list all matching products.
  Click Add to create a new product.
Click Settings > All Products Settings to display and edit settings that apply to all products. See All Products Settings for more details.
  The Filter dropdown controls which products are displayed on this page. The default is Products. You can choose to show only products with Low Inventory or those Queued for PO. For more on this, see Stock, Backorders, and Reordering.

The Product Edit Page

The Product edit page is where you manage all aspects of your products, from images to inventory control.

Click to create a new product. You'll be asked if you want to pre-fill all the fields with the data being shown.
Click to delete the current product.
Click Share to post this product to your Facebook and Twitter accounts. See Posting to Facebook and Twitter From Within Your Store.

See Learn More for information on all aspects of setting up and managing products.
